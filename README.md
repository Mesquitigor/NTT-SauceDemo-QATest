## Automação de testes - Sauce Demo

Esse repositório detém a automação de testes, feita em Python com o framework Robot Framework, referente à página https://www.saucedemo.com/. 
Todo código está compreensivo e para executá-lo, o comando utilizado é: **robot --outputdir C:\Seu\Diretorio\De\Evidencias\ --timestampoutputs tests.robot testtemplate_login.robot**.
